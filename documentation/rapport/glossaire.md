# 📖 Glossaire Technique — PFE IoT Security

| Terme | Définition |
|-------|-----------|
| **IoT** | Internet of Things — réseau d'objets connectés |
| **MQTT** | Message Queuing Telemetry Transport — protocole de messagerie léger pour IoT |
| **CoAP** | Constrained Application Protocol — protocole RESTful pour appareils contraints |
| **TLS** | Transport Layer Security — protocole de chiffrement des communications |
| **mTLS** | Mutual TLS — authentification mutuelle client/serveur via certificats |
| **DTLS** | Datagram TLS — TLS adapté aux protocoles UDP (CoAP) |
| **PKI** | Public Key Infrastructure — infrastructure de gestion des clés et certificats |
| **CA** | Certificate Authority — autorité de certification |
| **SIEM** | Security Information and Event Management |
| **IDS** | Intrusion Detection System |
| **IPS** | Intrusion Prevention System |
| **E2E** | End-to-End — de bout en bout |
| **DMZ** | Demilitarized Zone — zone réseau semi-protégée |
| **VPN** | Virtual Private Network |
| **ML** | Machine Learning |
| **PFE** | Projet de Fin d'Études |
| **FST** | Faculté des Sciences de Tunis |
| **TT** | Tunisie Telecom |
